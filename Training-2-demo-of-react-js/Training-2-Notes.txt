9)  Structure of App.jsx , importing the App.css, writing javascript funcion and displaying in html return.
    Example :
        import './App.css'

        function App() {
          let a = 10;

          return (
            <>
            <h1>The value of a : {a}</h1>
            </>
          )
        }

        export default App

********Step-3*******************
Calling One Component from another
a) Make a new component  
   i) define the compoent example :
   const Demo = () => {
    return <>
        <h1>I am Demo</h1>
    </>

    ii) Export the component name , example :
    export default Demo

  b) Call component in another component.
    i) Import the Demo.jsx component in App.jsx Example:
    import Demo from './Demo'

    ii) Call the Component name in Return of App.jsx , Example :
     return (
    <>
     <Demo/> 
     
    </>
  )
